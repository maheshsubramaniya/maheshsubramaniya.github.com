Installation & Running

Install Jetty
source: http://jekyllrb.com/docs/installation/


```
brew install ruby
gem install jekyll
gem install jekyll-sitemap
gem install rouge 
gem install rdiscount
```

Running:

```
jekyll serve
```
or
```
bundle exec jekyll serve
```
