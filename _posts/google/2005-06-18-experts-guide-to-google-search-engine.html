Today i had found this link, which exposes the various services provided by google search engine…note it, through search engines. it gives a list of commands to be used for quicker search.

Eg:

cache: Display Google’s cached version of a web page.

cache:www.googleguide.com(Show Google’s cached version of Google Guide’s home page.)

info: (or id:) Find info about a page. info:www.googleguide.com

(Find information about the Google Guide website.)

related: List web pages that are similar or related to the URL. related:www.googleguide.com (Find websites related to the Google Guide website.) RESTRICT SEARCH TO SITES WHERE QUERY WORDS APPEAR

allinanchor: All query words must appear in anchor text of links to the page. allinanchor: S-and-S Coupling (Search for pages in which the anchor text contains “S-and-S coupling”.)

inanchor: Terms must appear in anchor text of links to the page.

restaurants inanchor:gourmet (Search for pages on restaurants for which links to the page mention the word “gourmet”.)

Look at: http://www.googleguide.com/advanced_operators_reference.html
